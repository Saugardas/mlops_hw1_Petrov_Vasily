# mlops_hw1_Petrov_Vasily
ДЗ 1. Создание воспроизводимого ML-pipeline с использованием DVC и MLflow

В качестве ml-задачки будем использовать домашнюю работу 2го семестра по предмету "Основы машинного обучения", где мы классифицировали ирисы Фишера.

Для хранения датасета используется S3 хранилище Cloudru (см .dvc/config)

Инструкция по запуску:
Склонировать проект:
`git clone git@github.com:Saugardas/mlops_hw1_Petrov_Vasily.git`

(Опционально) создать новое окружение (например через conda):

`conda create -n mlops_hw python=3.12`
`conda activate mlops_hw`

Установить зависимости:
`pip install -r requirements.txt`

Чтобы забрать датасет, потребуется установить значения ключей access_key_id и secret_access_key (переданы в комментариях)

Дальше - загрузка датасета
`dvc pull`

Запуск:
`dvc repro --force` - force для того чтобы эксперимент запустился заново, несмотря на то что файлы не изменились. Или можно изменить параметры в params.yaml

Просмотр результатов логирования:
`mlflow ui --backend-store-uri sqlite:///mlflow.db`

По адресу http://127.0.0.1:5000 можно будет увидеть эксперименты
